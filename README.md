
## HOW TO USER
#### 1. create app_config.js and place this code
```javascript
module.exports = {
  USERNAME_KMUTNB: "username",      //ชื่อผู้ใช้
  PASSWORD_KMUTNB: "password",      //รหัสผ่าน
  JOB_SCHEDULE: '0 10 4 * * Monday' //วันที่
};
```
#### 2. config username and password
#### 3. run ```app.js```
